
# Instruction

- Los controles principales para controlar al 'Black_Hole' son:
    - 'W' || 'w' : para subir.
    - 'A' || 'a' : para ir a la izquierda.
    - 'S' || 's' : para bajar.
    - 'D' || 'd' : para ir a la derecha.
    - Mouse:
        - Apuntar : apuntar para posteriormente atacar.
        - Click : para atacar.
